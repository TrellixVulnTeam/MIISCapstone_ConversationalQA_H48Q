# BERT-FlowDelta

## Setup

Install all depedent python packages

```
pip install -r requirements.txt
```

Download necessary data

```
sh download.sh
```

## Train

```
sh train_quac.sh
```
## Test

```
sh test_quac.sh
```

please refer to our CodaLab submission pages for inference details and pretrained models
